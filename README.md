Car Price Prediction

Overview
A machine learning web application that predicts used car prices based on features like fuel type, transmission, kilometers driven, and ownership.

 Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

Features
- Car price prediction
- Interactive UI
- ML model integration
- Data preprocessing

Dataset
Used car dataset for training and prediction.
